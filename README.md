# Vintage Shirt Selling App - UI/UX Design Concept
A mobile shopping application UI/UX concept and interactive prototype designed in Figma.
# Project Overview
This project is a mobile application design concept for a Vintage Shirt E-commerce platform. Developed during my freshman year, this project focuses on creating an intuitive shopping experience specifically for vintage clothing enthusiasts, where unique aesthetics and ease of navigation are key.

# Design 
Aesthetic: Clean and modern layout with a focus on high-quality product imagery.
User Journey: Streamlined the process from browsing unique vintage collections.
Interactive Prototyping: Built dynamic transitions in Figma to simulate a real-world application feel.

# Key Features (UI/UX)
Discovery Feed: Designed a visually appealing home screen to showcase featured vintage items and new arrivals.

Product Details: Detailed view including size, condition, and price, with a clear "Add to Cart" call-to-action.

Intuitive Navigation: Simple bottom navigation bar for quick access to the Home, Search, and Profile sections.

Seamless Flow: Interactive "Next" and "Back" transitions to demonstrate the logical flow of the shopping process.

# Tools Used
Figma: Used for wireframing, high-fidelity UI design, and interactive prototyping.
